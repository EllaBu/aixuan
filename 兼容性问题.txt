1、手机键盘弹起的时候，把页面内容顶上去，
    解决方案： document.body && (document.body.scrollTop = document.body.scrollTop);

2、IFrame 中fixed失效
    解决方案：
    在被引入的页面中设置
    html,body {
      width: 100%;
      height: 100%;
    }
    .box {
      width: 100%;
      height: 100%;
      overflow-y: auto;
      -webkit-overflow-scrolling: touch;
    }
